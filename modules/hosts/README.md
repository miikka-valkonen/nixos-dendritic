# Host Configurations

This directory contains configurations specific to individual machines (hosts).

## Hosts

- `laeppaeri` - Old gaming laptop, Intel CPU & iGPU, Nvidia 960MX dGPU (currently disabled)

## Directory Structure

Each host directory typically contains:

- `configuration.nix`: Main NixOS system configuration — defines `configurations.nixos.<name>` with theme and `module` listing NixOS imports.
- `home.nix`: Home Manager configuration — defines `configurations.homeManager.<name>` with the `module` listing Home Manager imports.
- `hardware-configuration.nix`: Hardware-specific settings (usually generated by `nixos-generate-config`).

## Variable Management

Host-specific values live inside each host's `configuration.nix` under
`configurations.nixos.<host>`.

Common examples:

- `user.username`
- `user.email`
- `externalDevices.audioPlayer`
- `stateVersion`

## Adding a New Host

1. Create a new directory here (e.g., `mkdir modules/hosts/<name>`)
2. Add `configuration.nix`, `home.nix`, and `hardware-configuration.nix` following the pattern of existing hosts
3. The dendritic architecture automatically discovers the new host configuration — no registration needed
