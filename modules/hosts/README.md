## Directory Structure

Each host directory typically contains:

- `configuration.nix`: Main NixOS system configuration — defines `configurations.nixos.<name>` with wallpaper, theme, and the `module` listing NixOS imports.
- `home.nix`: Home Manager configuration — defines `configurations.homeManager.<name>` with the `module` listing Home Manager imports.
- `hardware-configuration.nix`: Hardware-specific settings (usually generated by `nixos-generate-config`).

## Adding a New Host

1. Create a new directory here (e.g., `mkdir modules/hosts/<name>`)
2. Add `configuration.nix`, `home.nix`, and `hardware-configuration.nix` following the pattern of existing hosts
3. The dendritic architecture automatically discovers the new host configuration — no registration needed
